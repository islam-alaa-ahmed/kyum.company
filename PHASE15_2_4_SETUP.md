# Phase 15.2.4 Setup

No SQL or Edge Function is required.

1. Copy the modified files into the repository.
2. Commit and push to GitHub.
3. Run `Ctrl + F5`.
4. Open `التقارير والتحليلات → مركز التقارير`.
5. Click `مركز التصدير`.
6. Test Excel, PDF, PNG and CSV.
7. Confirm that exported files include the active filters and hide UI controls.
