# Phase 15.3.3 Setup

No SQL or Edge Function is required.

1. Copy the modified files into the repository.
2. Commit and push to GitHub.
3. Run `Ctrl + F5`.
4. Open `إدارة العملاء → العملاء`.
5. Click `عرض`.
6. Verify health score, risk reasons, priority and recommended next action.
7. Test customers with no follow-ups, overdue follow-ups and accepted quotations.
