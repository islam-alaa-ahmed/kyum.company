# Phase 14.4 Setup

No SQL or Edge Function is required.

1. Copy the modified files into the repository.
2. Commit and push to GitHub.
3. Run `Ctrl + F5`.
4. Open Settings & Privacy → System Monitoring.
5. Click `تشغيل الفحص الشامل`.
6. Review Passed, Warning and Critical results.
7. Download JSON or HTML when a diagnostic report is required.
