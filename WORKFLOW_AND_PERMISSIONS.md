# Workflow & Permissions

## الأدوار

### Super Admin
إدارة جميع البيانات والمستخدمين والأدوار والقوائم المرجعية وسجل التدقيق.

### Sales Manager
يرى جميع العملاء والمندوبين والتقارير ويدير العملاء والمتابعات والعروض.

### Sales Representative
يرى ويدير العملاء والمتابعات والعروض المرتبطة به فقط.

### Viewer
عرض فقط دون إضافة أو تعديل أو حذف.

## دورة العميل

```text
إضافة عميل
   ↓
تحديد المندوب
   ↓
تسجيل متابعة
   ↓
إنشاء عرض سعر
   ↓
مراجعة العرض
   ├── مقبول → أمر بيع
   ├── مرفوض → سبب عدم البيع
   └── مؤجل → متابعة قادمة
```

## حالات عرض السعر

```text
تحت التجهيز
   ↓
تم الإرسال
   ↓
تحت المراجعة
   ├── مقبول
   ├── مرفوض
   └── ملغي
```

## قواعد العمل

- رقم الجوال لا يتكرر.
- رقم عرض السعر لا يتكرر.
- كل متابعة وعرض سعر مرتبطان بعميل.
- المندوب لا يرى أو يعدل بيانات مندوب آخر.
- حذف العميل للإدارة فقط.
- كل عملية أساسية تسجل في Audit Log.
