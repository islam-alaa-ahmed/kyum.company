# Phase 15.3.4 Setup

No SQL or Edge Function is required.

1. Copy the modified files into the repository.
2. Commit and push to GitHub.
3. Run `Ctrl + F5`.
4. Open a customer through `إدارة العملاء → العملاء → عرض`.
5. Click `تصدير الملف`.
6. Test PDF, Excel, PNG and direct print.
7. Verify that the file name contains the customer name and date.
