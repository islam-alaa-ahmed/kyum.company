1. Run `supabase/migrations/phase16_5_daily_alerts_escalations.sql`.
2. Copy the modified files to the same repository paths.
3. Commit and Push origin.
4. Run `Ctrl + F5`.
5. Open `إدارة المهام اليومية`.
6. Click `تحديث التنبيهات`.
7. Test Start, Escalate, Close and Reopen.
8. Open `التقارير والتحليلات → تقرير الأداء اليومي`.
9. Verify the daily alerts report per employee.
