# Phase 16.6 Setup

1. Run:
   `supabase/migrations/phase16_6_daily_attendance_activity_timeline.sql`
2. Copy the modified files to the same repository paths.
3. Commit and Push origin.
4. Run `Ctrl + F5`.
5. Sign out and sign in to start a fresh daily session.
6. Open `إدارة المهام اليومية` and verify the current session card.
7. Test `إنهاء يوم العمل`.
8. Open:
   `التقارير والتحليلات → تقرير الأداء اليومي`
9. Verify attendance summary, employee filter, activity-type filter and timeline.
