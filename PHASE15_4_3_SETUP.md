# Phase 15.4.3 Setup

No SQL or Edge Function is required.

1. Copy the modified files into the repository.
2. Commit and push to GitHub.
3. Run `Ctrl + F5`.
4. Open `إدارة العملاء → البيانات المرجعية`.
5. Use the dropdown to switch between the two sections.
6. Test Edit and Delete.
7. Confirm deletion is blocked when an item is already used in customer data.
