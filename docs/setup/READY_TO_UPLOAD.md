# READY TO UPLOAD

This package already contains:

- Supabase Project URL
- Supabase Publishable Key
- Supabase Auth integration
- Persistent session support
- User profile and role loading

No manual code editing is required before uploading to GitHub.

Important:
- Create the first user in Supabase Authentication.
- Promote that user to `super_admin`.
- Configure the GitHub Pages URL in Supabase Authentication URL Configuration.
