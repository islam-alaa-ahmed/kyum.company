# Phase 15.4.2 Setup

No SQL or Edge Function is required.

1. Copy the modified files into the repository.
2. Commit and push to GitHub.
3. Run `Ctrl + F5`.
4. Open `إدارة العملاء → مندوبو المبيعات`.
5. Test:
   - Search and status filter.
   - Edit.
   - Suspend and reactivate.
   - Delete a representative with no assigned customers.
   - Confirm deletion is blocked for representatives with assigned customers.
