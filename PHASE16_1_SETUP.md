# Phase 16.1 Setup

1. Run `supabase/migrations/phase16_1_daily_operations_center.sql` once.
2. Copy the modified files to the same repository paths.
3. Commit and Push origin.
4. Run `Ctrl + F5`.
5. In Permissions, grant `تحديث الإعلانات اليومي` → Edit to the roles allowed to complete it.
6. Sign in as a sales representative and verify `إدارة المهام اليومية` opens as the default screen.
7. Verify today customers, today follow-ups and overdue follow-ups.
